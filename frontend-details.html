What You'll Learn

Course Summary

Learn how to build Single Page Applications in various Front End Frameworks! In this course, you'll learn how to create both an Angular application and an Ember app from scratch. As you work through the course, you'll learn key architectural design techniques that make frameworks incredibly powerful.

Why Take this Course ?

In this course, you’ll explore popular JavaScript frameworks that make building complex web applications a lot easier! Follow along as you build interactive, single-page applications that work seamlessly in the browser without ever reloading the page. Filled with deep dives, code examples, and practical advice, you’ll uncover the magic behind JavaScript frameworks and what makes each of them unique.

Syllabus

Month 1
HTML 5 & CSS3
This course starts at the very beginning, even if most of the basics will be review. In your first week, through a mix of lessons and exercises, you’ll start learning HTML5 and CSS3. For this week’s project, you’ll build a beautiful single-page website for your new career, inspired by some of the best websites around. You’ll also select your domain name and start thinking about your new brand. Over the weekend, you’ll use FTP to take your site live. You’ll also write your first technical blog post.
Month 2
HTML 5 & CSS3
In month 2, we’ll continue learning HTML5 and CSS3. Your project this week will involve building a three-page website with a great-looking and functional form (like a contact form), embedded media and a navigation menu.
Month 3
Responsive Design
In moth 3, you’ll learn what it’s like to work with a designer through a mock client project. You’ll receive a website design for a single-page website and it’ll be your job to code it and, most importantly, make it responsive. This week, in addition to learning how to make sites work on any screen size, you’ll also learn about cool CSS3 techniques like transitions and animations, working with grids, CSS preprocessors like Sass and LESS, and much more. You’ll write another technical blog post over the weekend.
Month 4
JavaScript & jQuery
This month, we’ll split into two groups of 10 students (with two instructors designated per group) and dive into JavaScript & jQuery. We’ll start at the very beginning, by introducing you to programming fundamentals such as objects, functions, and arrays. For your first JavaScript project, you’ll work in pairs to create an interactive user experience using jQuery gallery. It’s time to let your creativity shine
Month 5
JavaScript & jQuery
In our second month tackling JavaScript & jQuery, you’ll also learn about Ajax, JSON and working with APIs. Your final project for the month will be a one-page web app. You’ll use an API to pull in data, and then add some functionality and style the rest of the site. Over the week, you’ll write a technical blog post about the process of creating your web app.
Month 6
JavaScript & jQuery
In our final month dedicated to JavaScript & jQuery, you’ll learn about version control and open-sourcing code using Git and Github, as well as workflow automation and Javascript tooling with Gulp. For this week’s project you’ll be assigned to a team of three and given a fake client API project, complete with technical requirements and a style guide. In your group, you’ll use your knowledge to develop the app from scratch, while collaborating as a team both in person and via Git and GitHub.
Month 7
WordPress
This month, we’ll dive into WordPress. Starting from the beginning, you’ll learn all about how you can use WordPress to create awesome CMS-powered websites for your clients. As your final project for the week, you’ll create a custom WordPress theme for a specific industry, like a microbrewery or salon. You’ll then get it up on a production server.
Month 8
WordPress
This month, you’re going to learn about something that brings real power and unlimited functionality to WordPress – custom post types and custom fields. You’ll spend this month and month 9 working on your personal portfolio site, powered by WordPress. It will give you the chance to show off all the awesome projects you developed over the past eight months!
Month 9
WordPress
This month will be all about the students – what do you need in order to get ready for Demo Day? Though there will be a few lessons and exercises on more advanced topics related to WordPress, jQuery and JavaScript, you’ll have a lot of time to get help with exactly what you need in order to be ready for Demo Day, Industry Day and for the following month – your first month as a front-end web developer.
